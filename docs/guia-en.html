Gecko User Guide

This guide explains how to use Gecko step by step: from choosing how to install it to accessing your services from anywhere in the world.

🛣️ Choose how you want to install Gecko

Three paths, the same open-source code. They all do the same thing once they are up and running.

💿
Gecko OS USB ★ RECOMMENDED
Bootable image for a dedicated mini PC. You flash a USB drive with Etcher, plug it into the PC + TV, and run the wizard. The mini PC stays dedicated to Gecko.
🪟
Windows installer
A ~25 MB .exe. Installs Gecko as a Windows service. Requires Docker Desktop. The UI opens in your default browser.
🍎
macOS installer
A .pkg for Apple Silicon or Intel. Installs Gecko as a daemon (launchd). Requires Docker Desktop. UI in the browser.
💡
Not sure which one to pick? If you want a dedicated device that is always on, go with Gecko OS. If you already have a PC that is always on, use the installer for your OS. The best "Netflix on the TV" experience is Gecko OS with a mini PC.

🧙 The installation assistant

The wizard adapts automatically to the path you chose. You will see between 5 and 8 steps depending on your system.

1

Welcome + system checks

Introductory screen with a summary of what Gecko does. Below it, a discreet grid shows your system's RAM, CPU, free disk; if something is below the recommended level, you will see a yellow warning. It does not block, it only informs. Press Start.

2

WiFi (Gecko OS only, no cable)

If you plugged it in via ethernet, this step is skipped. If you need to use WiFi, Gecko scans the available networks and lets you choose one and enter the password. There is an "I have a cable" button to skip if you prefer.

3

Docker check (Windows/macOS only)

Gecko checks whether Docker Desktop is active. If you don't have it, you will see a button to download it. On Gecko OS, Docker comes preinstalled and active, so this step does not appear.

4

Data folder / disk

Where the files you download will be stored. We recommend an external USB-3.0 disk of at least 1 TB. You need at least 100 GB free. Gecko does not delete anything that is already there.

5

Admin password

A single password for everything: Jellyfin admin, qBittorrent, dashboard. Minimum 6 characters. Write it down, you won't be able to recover it if you lose it.

6

Subtitle language

Choose one of the 3 pre-seeded profiles in Bazarr: "Català + English", "Castellano + English" or "English only". Bazarr will automatically download subtitles in these languages.

7

VPN (practically mandatory)

In DE/FR/UK/US, without a VPN you get warnings from your ISP within weeks. In ES/IT it is less enforced but the risk is there. We recommend ProtonVPN (it has a free plan without P2P, ~4€/month with P2P) or Mullvad (~5€/month, accepts cash). You can skip and set it up later.

✓

Installation + auto-configuration

Gecko downloads the Docker images (~3 GB the first time) and configures them with each other: Prowlarr↔arrs, qBittorrent to the arrs, Jellyseerr↔Jellyfin+arrs, Bazarr↔Sonarr+Radarr. It can take 5-10 min. Once it finishes, the only manual step left is to add indexers to Prowlarr.

✅
Almost fully automatic config: Gecko connects all the services automatically, seeds TRaSH-Guides quality profiles, seeds subtitle profiles. The only manual step left is to add indexers to Prowlarr (the legal line we draw). How to do it →
💾
Gecko OS only: Once the installation is finished and everything works, you can go to Settings → Move to internal disk to copy Gecko to the mini PC's internal disk. After that you can remove the USB. ~5 min, automatic reboot, idempotent.

🗂️ The dashboard side menu

Once the stack is installed, Gecko always shows the dashboard with a navigation sidebar on the left containing four sections:

🏠
Home
Stack status (running / stopped), buttons to start and stop all the services, and quick one-click access to each service.
🔒
VPN
See whether the VPN is active. Enable ProtonVPN or Mullvad by entering the WireGuard keys. Disable it if needed. Changes are applied by automatically restarting the stack.
🌐
Network
Your local IP on the Wi-Fi network, the URLs of each service for the local network, and information to access from outside your home with Tailscale.
⬇️
Downloads
See the in-progress downloads from Radarr and Sonarr in real time: progress, size and time remaining. For each item you can do a blocklist + new search if the download isn't what you want.
⚙️
Settings
Change the app language (Catalan, Spanish, English). See the Gecko version. Reinstall the stack from scratch (it does not delete the media content).

Language: At the bottom of the sidebar there are the CA / ES / EN buttons to change the language at any time.

🏠 Home page, Starting the services

This is the dashboard's main screen. Here you can:

  • See the status: a colored indicator shows whether the stack is Running (green), Stopped (red) or Checking (yellow).
  • Start the stack: press the Start button. The Docker services start in the background. It can take 20-60 seconds the first time it starts.
  • Stop the stack: the Stop button stops all the Docker containers. Useful to save resources when it's not in use.
  • Access a service: the six buttons in the grid open the service directly in the browser (localhost URLs). They are disabled if the stack is stopped.
⚠️
Services are accessed via localhost from the computer itself. To access from other devices (phone, TV, laptop), you need to use the local network IP or the Tailscale IP. See the Network section.

🔒 VPN page

From here you can manage the VPN at any time, without having to reinstall.

Which providers work?

Gecko uses Gluetun as the VPN container. Not all VPN providers work with this setup; they need to offer WireGuard with a private key and an assigned IP address. The two that work in a simple way:

  • Mullvad: the simplest. Generate a key directly on the website without downloading files. ~5€/month without a subscription.
  • ProtonVPN: generates a WireGuard configuration file from which the data is extracted. It has a free plan.
⚠️
NordVPN, Surfshark and similar ones do not work directly with this method. Gluetun supports them but it requires specific environment variables for each provider, beyond the scope of the assistant.

Mullvad, how to get the data

1

Log in to my.mullvad.net

Go to my.mullvad.net and access your account.

2

WireGuard configuration → Add new device

In the WireGuard configuration section, type a name (e.g. gecko) and click Generate key.

3

Copy the private key and the address

Mullvad will show the private key (a long string) and the assigned IP address (10.68.x.x/32). Copy both into Gecko and press Enable VPN.

ProtonVPN, how to get the data

1

Go to account.proton.me → VPN → Downloads

Log in to account.proton.me and navigate to VPN → Downloads → WireGuard configuration.

2

Generate and download the .conf file

Choose a country/server and download the .conf configuration file.

3

Extract PrivateKey and Address from the file

Open the .conf with Notepad. In the [Interface] section you will find the two pieces of data you need:

[Interface]
PrivateKey = ABC123...xyz ← private key
Address = 10.2.x.x/32 ← assigned IP address

Copy both values into Gecko and press Enable VPN.

Disable the VPN

Press the Disable VPN button. Gecko removes the keys and switches to the no-VPN compose automatically.

🔌 Set up Prowlarr, the only manual step

Once the Gecko assistant is finished, the only thing you have to do manually is add indexers to Prowlarr. An indexer is a content search engine (like Google but for movies and TV shows). Without any indexer, Radarr and Sonarr won't find anything.

ℹ️
Prowlarr is already connected to Radarr, Sonarr and Lidarr. Gecko has set this up automatically. When you add an indexer to Prowlarr, all the services will see it instantly, no need to do anything else.

To add indexers to Prowlarr:

1

Open Prowlarr

Go to localhost:9696 or click the Prowlarr button from the Gecko dashboard.

2

Indexers → Add Indexer

Search for and add the indexers you want. Prowlarr has hundreds of public and private sources available.

✓

You're all set

The indexers are applied automatically to Radarr, Sonarr and Lidarr. Now you can request content from Jellyseerr or add movies and TV shows directly from Radarr/Sonarr.

How to find good indexers

  • Search for "best public Prowlarr indexers [current year]" on Google; there are lists that are updated regularly.
  • Ask an AI (ChatGPT, Gemini…): "What are the best public Prowlarr indexers for movies and TV shows in Catalan/Spanish?"
  • Check the official Prowlarr documentation to see the full catalog.
⚠️
Gecko does not include or recommend any specific indexer. The user is solely responsible for the content they access and for complying with the laws in force in their country.

📶 Local access vs. external access

There are two ways to access your services, depending on where you are:

Situation URL to use Works? Needs Tailscale?
At home, on the same Wi-Fi http://192.168.1.X:PORT ✓ Yes No
At home, from the PC itself http://localhost:PORT ✓ Yes No
Away from home, phone on mobile data http://192.168.1.X:PORT ✗ No Yes, required
Away from home, another Wi-Fi network http://192.168.1.X:PORT ✗ No Yes, required
With Tailscale, from anywhere http://100.X.X.X:PORT ✓ Yes Yes (already set up)

🔗 Set up Tailscale (external access)

Tailscale creates a virtual private network between all your devices. Once it's set up, you can access your Gecko services from anywhere in the world, as if you were at home. There's no need to open ports on the router or have a public IP.

✅
Tailscale is free for personal use (up to 3 users and 100 devices). There is no speed or data limit.
1

Install Tailscale on the computer running Gecko

Download Tailscale for Windows, install it and log in with Google, Microsoft or email. Tailscale runs in the background.

2

Note down your computer's Tailscale IP

In the taskbar, click the Tailscale icon and you will see your Tailscale IP; it has the format 100.x.x.x. Note it down.

3

Install Tailscale on the other devices

Download Tailscale on your phone (iOS / Android), laptop or tablet. Log in with the same account. Turn on the connection.

4

Access the services with the Tailscale IP

From any device with Tailscale active, use the IP 100.x.x.x instead of the local IP.

How the URLs look with Tailscale

If your Tailscale IP is 100.64.1.25, the URLs would be:

# Jellyfin (player)
http://100.64.1.25:8096

# Jellyseerr (requests)
http://100.64.1.25:5055

# Radarr · Sonarr · Prowlarr · qBittorrent
http://100.64.1.25:7878 ← Radarr
http://100.64.1.25:8989 ← Sonarr
http://100.64.1.25:9696 ← Prowlarr
http://100.64.1.25:8090 ← qBittorrent
💡
Tip: Save the Tailscale URLs in your phone browser's bookmarks for quick access. The Tailscale URLs always work, whether you are at home or away.

Tailscale and Gecko's VPN

Tailscale and Gecko's download VPN (ProtonVPN/Mullvad via Gluetun) do not interfere with each other. Tailscale is for accessing the services remotely. Gecko's VPN is so that the qBittorrent downloads go out encrypted. You can have both active at the same time without any problem.
