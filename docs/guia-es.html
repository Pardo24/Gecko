🛣️ Elige cómo quieres instalar Gecko

Tres caminos, el mismo código abierto. Todos hacen lo mismo una vez están en marcha.

💿
Gecko OS USB ★ RECOMENDADO
Imagen booteable para un mini PC dedicado. Flasheas un USB con Etcher, lo enchufas al PC + TV, y haces el wizard. El mini PC queda dedicado a Gecko.
🪟
Instalador Windows
.exe de ~25 MB. Instala Gecko como servicio de Windows. Necesita Docker Desktop. La UI se abre en tu navegador por defecto.
🍎
Instalador macOS
.pkg para Apple Silicon o Intel. Instala Gecko como daemon (launchd). Necesita Docker Desktop. UI en el navegador.
💡
¿No sabes cuál elegir? Si quieres un dispositivo dedicado siempre encendido, Gecko OS. Si ya tienes un PC siempre encendido, el instalador de tu OS. La mejor experiencia "Netflix en la TV" es Gecko OS con mini PC.

🧙 El asistente de instalación

El wizard se adapta automáticamente al camino que has elegido. Verás entre 5 y 8 pasos dependiendo de tu sistema.

1

Bienvenida + comprobaciones del sistema

Pantalla introductoria con un resumen de qué hace Gecko. Debajo, una rejilla discreta muestra RAM, CPU, disco libre de tu sistema; si algo está por debajo de lo recomendado, verás un aviso amarillo. No bloquea, solo informa. Pulsa Empezar.

2

WiFi (solo Gecko OS, sin cable)

Si lo has enchufado por ethernet, este paso se salta. Si tienes que usar WiFi, Gecko escanea las redes disponibles y te deja elegir y poner contraseña. Botón "Tengo cable" para saltar si quieres.

3

Comprobación de Docker (solo Windows/macOS)

Gecko comprueba si Docker Desktop está activo. Si no lo tienes, verás un botón para descargarlo. En Gecko OS, Docker ya viene preinstalado y activo, este paso no aparece.

4

Carpeta de datos / disco

Donde se guardarán los archivos que descargues. Recomienda un disco externo USB-3.0 de mínimo 1 TB. Necesitas al menos 100 GB libres. Gecko no borra nada de lo que ya haya.

5

Contraseña admin

Una sola contraseña para todo: Jellyfin admin, qBittorrent, dashboard. Mínimo 6 caracteres. Apúntala, no la podrás recuperar si la pierdes.

6

Idioma de los subtítulos

Elige uno de los 3 perfiles presembrados en Bazarr: "Català + English", "Castellano + English" o "English only". Bazarr descargará subtítulos automáticamente en estos idiomas.

7

VPN (prácticamente obligatoria)

En DE/FR/UK/US sin VPN tienes avisos del ISP en semanas. En ES/IT está menos aplicado pero el riesgo existe. Recomendamos ProtonVPN (tiene plan gratuito sin P2P, ~4€/mes con P2P) o Mullvad (~5€/mes, acepta efectivo). Puedes saltarlo y configurarla después.

✓

Instalación + autoconfiguración

Gecko descarga las imágenes Docker (~3 GB la primera vez) y las configura entre ellas: Prowlarr↔arrs, qBittorrent en los arrs, Jellyseerr↔Jellyfin+arrs, Bazarr↔Sonarr+Radarr. Puede tardar 5-10 min. Una vez termine, el único paso manual que queda es añadir indexers a Prowlarr.

✅
Autoconfiguración casi total: Gecko conecta todos los servicios automáticamente, siembra perfiles de calidad TRaSH-Guides, siembra perfiles de subtítulos. El único paso manual que queda es añadir indexers a Prowlarr (la línea legal que dibujamos). Cómo hacerlo →
💾
Solo Gecko OS: Una vez terminada la instalación y todo funciona, puedes ir a Settings → Move to internal disk para copiar Gecko al disco interno del mini PC. Después puedes retirar el USB. ~5 min, reinicio automático, idempotente.

🗂️ El menú lateral del dashboard

Una vez instalado el stack, Gecko muestra siempre el dashboard con una barra lateral de navegación a la izquierda con cuatro secciones:

🏠
Inicio
Estado del stack (en marcha / parado), botones para arrancar y parar todos los servicios, y acceso rápido a cada servicio con un clic.
🔒
VPN
Ver si la VPN está activa. Activar ProtonVPN o Mullvad introduciendo las claves WireGuard. Desactivarla si es necesario. Los cambios se aplican reiniciando el stack automáticamente.
🌐
Red
Tu IP local en la red Wi-Fi, las URLs de cada servicio por red local, e información para acceder desde fuera de casa con Tailscale.
⬇️
Descargas
Ve en tiempo real las descargas en curso de Radarr y Sonarr: progreso, tamaño y tiempo restante. Para cada elemento puedes hacer blocklist + nueva búsqueda si la descarga no te interesa.
⚙️
Configuración
Cambiar el idioma de la aplicación (catalán, castellano, inglés). Ver la versión de Gecko. Reinstalar el stack desde cero (no borra el contenido multimedia).

Idioma: En la parte inferior de la barra lateral están los botones CA / ES / EN para cambiar el idioma en cualquier momento.

🏠 Página Inicio, Arrancar los servicios

Esta es la pantalla principal del dashboard. Aquí puedes:

  • Ver el estado: un indicador de color muestra si el stack está En marcha (verde), Parado (rojo) o Comprobando (amarillo).
  • Arrancar el stack: pulsa el botón Arrancar. Los servicios Docker se inician en segundo plano. Puede tardar 20-60 segundos la primera vez que se arranca.
  • Parar el stack: el botón Parar detiene todos los contenedores Docker. Útil para ahorrar recursos cuando no se usa.
  • Acceder a un servicio: los seis botones de la rejilla abren directamente el servicio en el navegador (URLs de localhost). Están desactivados si el stack está parado.
⚠️
A los servicios se accede vía localhost desde el propio ordenador. Para acceder desde otros dispositivos (móvil, televisión, portátil), hay que usar la IP de la red local o la IP de Tailscale. Consulta la sección Red.

🔒 Página VPN

Desde aquí puedes gestionar la VPN en cualquier momento, sin tener que reinstalar.

¿Qué proveedores funcionan?

Gecko utiliza Gluetun como contenedor VPN. No todos los proveedores VPN funcionan con esta configuración, es necesario que ofrezcan WireGuard con clave privada y dirección IP asignada. Los dos que funcionan de forma sencilla:

  • Mullvad: el más sencillo. Genera una clave directamente en la web sin descargar archivos. ~5€/mes sin suscripción.
  • ProtonVPN: genera un archivo de configuración WireGuard del cual se extraen los datos. Tiene plan gratuito.
⚠️
NordVPN, Surfshark y similares no funcionan directamente con este método. Gluetun los soporta pero requiere variables de entorno específicas para cada proveedor, fuera del alcance del asistente.

Mullvad, cómo obtener los datos

1

Inicia sesión en my.mullvad.net

Ve a my.mullvad.net y accede a tu cuenta.

2

WireGuard configuration → Add new device

En la sección WireGuard configuration, escribe un nombre (ej: gecko) y haz clic en Generate key.

3

Copia la clave privada y la dirección

Mullvad mostrará la clave privada (string largo) y la dirección IP asignada (10.68.x.x/32). Copia ambas en Gecko y pulsa Activar VPN.

ProtonVPN, cómo obtener los datos

1

Ve a account.proton.me → VPN → Downloads

Inicia sesión en account.proton.me y navega a VPN → Downloads → WireGuard configuration.

2

Genera y descarga el archivo .conf

Elige un país/servidor y descarga el archivo de configuración .conf.

3

Extrae PrivateKey y Address del archivo

Abre el .conf con el Bloc de notas. En la sección [Interface] encontrarás los dos datos que necesitas:

[Interface]
PrivateKey = ABC123...xyz ← clave privada
Address = 10.2.x.x/32 ← dirección IP asignada

Copia ambos valores en Gecko y pulsa Activar VPN.

Desactivar la VPN

Pulsa el botón Desactivar VPN. Gecko elimina las claves y pasa al compose sin VPN automáticamente.

IP de red local

Tu IP local (ej: 192.168.1.42) es la que usan los dispositivos conectados al mismo Wi-Fi o cable de casa. Para acceder a Jellyfin desde el móvil (en el mismo Wi-Fi), la URL sería http://192.168.1.42:8096.

🚫
La IP local NO funciona fuera de casa. Si intentas acceder a http://192.168.1.42:8096 desde el móvil con datos móviles, o desde otra red, no funcionará. Para acceder desde cualquier lugar hay que configurar Tailscale. Cómo configurar Tailscale →

🔌 Configurar Prowlarr, el único paso manual

Una vez terminado el asistente de Gecko, lo único que tienes que hacer manualmente es añadir indexadores a Prowlarr. Un indexador es un buscador de contenido (como Google pero para películas y series). Sin ningún indexador, Radarr y Sonarr no encontrarán nada.

ℹ️
Prowlarr ya está conectado a Radarr, Sonarr y Lidarr. Gecko lo ha configurado automáticamente. Cuando añades un indexador a Prowlarr, todos los servicios lo verán al momento, no hay que hacer nada más.

Para añadir indexadores a Prowlarr:

1

Abre Prowlarr

Ve a localhost:9696 o haz clic en el botón Prowlarr desde el dashboard de Gecko.

2

Indexers → Add Indexer

Busca y añade los indexadores que quieras. Prowlarr tiene cientos de fuentes públicas y privadas disponibles.

✓

Ya estás listo

Los indexadores se aplican automáticamente a Radarr, Sonarr y Lidarr. Ahora puedes pedir contenido desde Jellyseerr o añadir películas y series directamente desde Radarr/Sonarr.

Cómo encontrar buenos indexadores

  • Busca "best public Prowlarr indexers [año actual]" en Google, hay listas actualizadas regularmente.
  • Pregunta a una IA (ChatGPT, Gemini…): "¿Cuáles son los mejores indexadores públicos para Prowlarr para películas y series en catalán/español?"
  • Consulta la documentación oficial de Prowlarr para ver el catálogo completo.
⚠️
Gecko no incluye ni recomienda ningún indexador concreto. El usuario es el único responsable del contenido al que accede y de cumplir la legislación vigente en su país.

📶 Acceso local vs. acceso externo

Hay dos maneras de acceder a tus servicios, dependiendo de dónde estés:

Situación URL a usar ¿Funciona? ¿Necesita Tailscale?
En casa, en el mismo Wi-Fi http://192.168.1.X:PORT ✓ Sí No
En casa, desde el propio PC http://localhost:PORT ✓ Sí No
Fuera de casa, móvil con datos http://192.168.1.X:PORT ✗ No Sí, obligatorio
Fuera de casa, otra red Wi-Fi http://192.168.1.X:PORT ✗ No Sí, obligatorio
Con Tailscale, desde cualquier lugar http://100.X.X.X:PORT ✓ Sí Sí (ya configurado)

🔗 Configurar Tailscale (acceso externo)

Tailscale crea una red privada virtual entre todos tus dispositivos. Una vez configurado, puedes acceder a los servicios de Gecko desde cualquier lugar del mundo, como si estuvieras en casa. No hace falta abrir puertos en el router ni tener IP pública.

✅
Tailscale es gratuito para uso personal (hasta 3 usuarios y 100 dispositivos). No hay límite de velocidad ni de datos.
1

Instala Tailscale en el ordenador con Gecko

Descarga Tailscale para Windows, instálalo e inicia sesión con Google, Microsoft o correo electrónico. Tailscale se ejecuta en segundo plano.

2

Anota la IP de Tailscale de tu ordenador

En la barra de tareas, haz clic en el icono de Tailscale y verás tu IP de Tailscale, es del formato 100.x.x.x. Anótala.

3

Instala Tailscale en los demás dispositivos

Descarga Tailscale en el móvil (iOS / Android), portátil o tablet. Inicia sesión con la misma cuenta. Activa la conexión.

4

Accede a los servicios con la IP de Tailscale

Desde cualquier dispositivo con Tailscale activo, usa la IP 100.x.x.x en lugar de la IP local.

Cómo quedan las URLs con Tailscale

Si tu IP de Tailscale es 100.64.1.25, las URLs serían:

# Jellyfin (reproductor)
http://100.64.1.25:8096

# Jellyseerr (peticiones)
http://100.64.1.25:5055

# Radarr · Sonarr · Prowlarr · qBittorrent
http://100.64.1.25:7878 ← Radarr
http://100.64.1.25:8989 ← Sonarr
http://100.64.1.25:9696 ← Prowlarr
http://100.64.1.25:8090 ← qBittorrent
💡
Consejo: Guarda las URLs de Tailscale en los marcadores del navegador de tu móvil para acceder a ellas rápidamente. Las URLs de Tailscale siempre funcionan, tanto si estás en casa como fuera.

Tailscale y la VPN de Gecko

Tailscale y la VPN de las descargas de Gecko (ProtonVPN/Mullvad vía Gluetun) no interfieren entre ellas. Tailscale es para acceder remotamente a los servicios. La VPN de Gecko es para que las descargas de qBittorrent salgan cifradas. Puedes tener las dos activas simultáneamente sin ningún problema.
